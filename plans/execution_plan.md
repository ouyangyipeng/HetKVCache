## 二、执行步骤

### 阶段1: 环境验证与编译 (优先级: 最高)

#### 步骤1.1: 检查NPU状态
```bash
# 检查NPU设备状态
npu-smi info

# 检查CANN环境
cat /usr/local/Ascend/ascend-toolkit/latest/version.cfg

# 设置环境变量
source /usr/local/Ascend/ascend-toolkit/set_env.sh
```

#### 步骤1.2: 编译项目
```bash
cd /root/proj/HetKVCache
mkdir -p build && cd build
cmake -DUSE_ASCEND=ON ..
make -j$(nproc)
```

#### 步骤1.3: 运行单元测试
```bash
./hetkvcache_test
```

#### 步骤1.4: 运行Benchmark
```bash

```
